Welcome! This project aims to predict the outcome of NBA matches using Logistic Regression with features including ELO, 3-point field goal, offensive/defensive rating, and much more. Final accuracy is around 65.5%. The source code can be found at my Github repository

    Detailed description of ELO score calculation
  • Initially, each team has an ELO score of 1600
  • Assume team A's current ELO score is RA and team B's current ELO score is RB
  • The expected win rate of A is EA = 1/( 1 + 10^( (RB-RA)/400 ) )
  • Similarly, expected win rate of B is EB = 1/( 1 + 10^( (RA-RB)/400 ) )
  • SA = If team A wins the match, it gets 1 point, if tie it gets 0.5 point, if lose it gets 0 point
  • Adjust team A's ELO score: RA' = RA + K * (SA - EA). RA' is now the new ELO score. Do the same for team B
  • K is defined as 32 if ELO is below 2100, 24 if ELO is between 2100 and 2400, 16 if ELO is above 2400. In this particular project ELO never surpasses 2100, so K is always 32
